# Typography


The fonts used in AUX: Georgia, Oswald, Open Sans, Work Sans

Headings should be use the following font weights:
- `h1`: 900
- `h2`: 900
- `h3`: 500
- `h4`: Normal
- `h5`: Normal
- `h6`: Normal

## Georgia

**Font:** Georgia

**Usage:** Recommended for headings h1-h3. Georgia should **NOT** to be used for body copy.

## Work Sans

**Font:** Work Sans

**Weights:** 100, 200, 300, 400, 500, 600, 700, 800, 900

**Usage:** Recommended for headings h1-h3. Work Sans should **NOT** be use for large blocks of body copy. Work Sans is brought in more as a utility font on elements like forms and buttons.

**URL:** [https://fonts.google.com/specimen/Work+Sans](https://fonts.google.com/specimen/Work+Sans)
